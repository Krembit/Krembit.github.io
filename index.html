Key Terms

World Wide Web

The World Wide Web refers to the links between all of the files which constitute web pages. This is a distinct concept from the Internet.

Internet

The Internet refers to all of the computers around the world which are able to send data back and forth to each other, an example of data sent being web pages.

Web Browser

A Web Browser is software on your computer which establishes a link to another computer over the internet and requests web pages from it, and then displays them on your computer. Those web pages may contain links to other pages, forming a web of links between all the pages which link to each other.
