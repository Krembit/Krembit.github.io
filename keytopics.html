Key Topics

Software

Software, or a program, is a file on a computer which includes specific instructions that cause the computer to perform specific tasks.

Internet

A network is any number of computers which are all connected. Local Area Networks (LANs) are located very close together, usually in the same building. A Wide Area Network (WAN) is one which connects multiple LANs together. The Internet is a WAN which covers large parts of the planet and allows computers very far away from one another to communicate quickly.

Web Page

A web page is a file which exists on a computer somewhere that can be downloaded by your computer if your computer requests it. Web pages can include text, images, sounds, and videos, or whatever else the developer wants to include in it. Web Pages can also link to other web pages, allowing you to see content related to the content you're already seeing.

World Wide Web

When multiple web pages all link to one another, a map drawn of all the different pathways between them looks like a web. This is why the interconnected links between web pages is called the World Wide Web. Note that this concept is distinct from the Internet. While the internet refers to connections between computers, the World Wide Web refers to connections between web pages.

Web Server

A web server is any computer which contains the files making up a web page and can receive and fulfill requests for that page. The requesting computer sends a message to the server, which will package the page and send it over to be downloaded. When a user clicks a link on a web page, it sends another request to the server which then fulfills that as well.

Web Browser

A Web Browser is a program which is able to establish links with web servers, request web pages, download them, and display them for the user. Web browsers have many useful features other than this which aid the user in finding information they need and remembering where it was found.

Cache

Web browsers will usually have a feature called a cache. When the browser requests a web page for the first time, it will download the page and store some of the content from the page on your computer. Then, if you navigate to that page later, your computer can access the information which has been stored in the cache instead of having to request it from the server again, which speeds up the loading of the page.

HTML

HTML stands for HyperText Markup Language. This is the code that web developers write in order to put content onto a web page. It uses tags to specify how each element on the page should look. For example, the large, bold font on this page uses the header tag, which looks like <h#>, where # is the rank of the header. First rank (1) is the biggest header, and second, third, so on and so forth are all progressively smaller.
