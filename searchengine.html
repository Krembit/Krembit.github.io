Search Engines

Some might be confused by the term “Search Engine,” so I will inform you that they are a distinct concept from Web Browsers. Search engines are, in fact, websites, contained on servers just like any other. The function of a search engine is to take a query given by the user and look through many different web pages to find ones which contain references to the query. For example, typing “Why is the sky blue” into an unintegrated URL bar will simply return a message stating that it has no idea which website you want, typing that into a search engine will return results which will teach you about the refractive index of oxygen in the atmosphere. When configuring a modern browser, many will have you set a default search engine. This is because of the integrated URL bar. When you use it, you can either type a URL or a query, saving you a step in that you do not need to first navigate to google.com to figure out who won the Super Bowl. You can simply type the query into your browser’s URL bar and it will first figure out that this is Probably Not a URL and send it to your default search engine and allow it to figure out what you want. As well, if you are typing a URL into the URL bar and happen to type it incorrectly, it will helpfully navigate you to a page which will contain suggestions for what you may have meant to search for instead.
